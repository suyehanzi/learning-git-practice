# Learning Git Practice

This repository is used to practice Git and GitHub step by step.

## Project Structure

- `README.md`: repository introduction
- `docs/`: learning notes and GitHub practice guides
- `scripts/`: small practice scripts
- `notes/`: simple text files for push and pull practice

## Current Practice Goals

- create and review commits
- inspect history with `git log` and `git show`
- undo changes with `git restore`
- create, merge, and delete branches
- prepare the repository for GitHub

## Suggested Next Steps

1. Create a GitHub repository.
2. Add the remote with `git remote add origin <url>`.
3. Push the local `main` branch.
4. Keep editing files here to practice `push` and `pull`.
